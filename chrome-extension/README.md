# How to install

1. Open Chrome → go to chrome://extensions
2. Turn on "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the "chrome-extension" folder
5. Done

# How it works

1. Click "Send on WhatsApp" on any order
2. Receipt image copied to clipboard automatically
3. WhatsApp Web opens with customer's number
4. Extension auto-pastes receipt into message box
5. Click Send manually
